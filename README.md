# x-metrics (Archived)

> **This project has been archived.** It has been superseded by [crossplane-contrib/resource-state-metrics](https://github.com/crossplane-contrib/resource-state-metrics), a completely new project built on top of the upstream [resource-state-metrics (RSM)](https://github.com/kubernetes-sigs/resource-state-metrics) from Kubernetes SIG Instrumentation.

## Why?

Rather than continuing to build and maintain a parallel implementation, the Crossplane community decided to align with the upstream Kubernetes effort. See the [Crossplane Metrics Proposal (crossplane/crossplane#6865)](https://github.com/crossplane/crossplane/pull/6865) and the [decision comment](https://github.com/crossplane/crossplane/pull/6865#issuecomment-3767353784) for details.

Please head to [crossplane-contrib/resource-state-metrics](https://github.com/crossplane-contrib/resource-state-metrics) for the new project.
